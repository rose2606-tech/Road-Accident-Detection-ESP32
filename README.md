# 🚗 Road Accident Detection System using ESP32

## 📌 Overview
This project is an IoT-based road accident detection and monitoring system developed using ESP32, IR sensor, and PIR sensor. It detects potential accidents and sends real-time alerts via the Blynk IoT platform.

## ⚙️ Technologies Used
- ESP32
- Arduino IDE
- Blynk IoT Platform
- IR Sensor
- PIR Sensor
- Embedded C (Arduino)

## 🚨 Features
- Detects accidents using IR sensor
- Detects motion using PIR sensor
- Sends real-time alerts to mobile via Blynk
- Provides buzzer and LED alerts
- Multi-level alert system:
  - Critical Alert (Accident + Motion)
  - Accident Detection
  - Motion Detection
  - Normal Monitoring

## 🔌 Hardware Components
- ESP32
- IR Sensor
- PIR Sensor
- Buzzer
- LED

## 📲 Working
1. IR sensor detects obstacle/accident
2. PIR sensor detects motion
3. ESP32 processes sensor data
4. Alerts are sent via Blynk app
5. Buzzer and LED activate for warning

## ⚠️ Note
Sensitive information like WiFi credentials and Blynk Auth Token have been removed for security reasons.
